# fe-estate-log

부동산 임장(현장 방문) 결과를 기록하고 관리하기 위한
프론트엔드 학습 중심 프로젝트입니다.

📕 [English README](./README.md)

## 목적
- Vite + React 기반 프로젝트를 처음부터 직접 구성
- 번들러, 설정 파일, 프로젝트 구조에 대한 이해
- Trunk-Based Development 방식으로 개발 경험

## 기술 스택 (초기)
- React 18
- TypeScript
- Vite

## 범위
- 임장 기록 작성 및 관리
- 지역 기반 게시글 검색
- 평점 및 속성 정보 저장

## 백엔드
- Spring Boot (프론트엔드와 분리된 레포지토리)
